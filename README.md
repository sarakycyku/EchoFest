# EchoFest

## Ekzekutimi i programit

Duhet te keni te instaluar nje server lokal si p.sh XAMPP.

Hap CMD dhe shkruaj

``git clone https://github.com/sarakycyku/EchoFest.git``

Projekti duhet te ruhet ne folderin htdocs

Starto Apache ne XAMPP Control Panel.
 
 Hap browser dhe shkruaj:

``http://localhost/EchoFest/pages/index.php``

ku hapet landing page dhe perdoruesi per te vazhduar me shume duhet ose te behet login ose signup.Dhe pasi qe perfundon kjo ridrejtohet ne home.php
